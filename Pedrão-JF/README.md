# Sistema de Conforto Térmico (IA baseada em regras)

Projeto desenvolvido para a disciplina de Inteligência Artificial.

## Descrição

Este sistema utiliza um conjunto de regras simbólicas para classificar condições de conforto térmico com base em temperatura e umidade relativa do ar.

## Tecnologias

- Python
- Pandas
- Matplotlib

## Execução

Instalar dependências:

pip install -r requirements.txt

Rodar o sistema:

python src/main.py

## Estrutura

- data → dados de entrada
- outputs → resultados gerados
- src → código principal
